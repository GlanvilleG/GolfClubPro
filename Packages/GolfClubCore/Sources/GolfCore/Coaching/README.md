
# Coaching

The Coaching subsystem provides personalised shot execution advice.

It consumes:

- RecommendationDecision
- ShotContext
- PlayingCharacteristics
- CoachingConfiguration

It produces:

- ShotPreparationAdvice

Responsibilities:

- Setup advice
- Alignment
- Ball position
- Tempo reminders
- Swing thoughts
- Confidence reinforcement
- Course management reminders

The Coaching subsystem does not determine club selection.

Recommendation determines *what* shot to play.

Coaching determines *how* to execute the shot.
