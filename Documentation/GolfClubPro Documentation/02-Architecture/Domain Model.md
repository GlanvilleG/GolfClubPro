
# Domain Model

## Purpose

The Domain Model defines the core business concepts used by GolfClubPro and GolfCore.

It describes what the system understands about golf before considering UI, database, APIs, or Apple frameworks.

## Core Domain Concepts

### Player

A golfer using GolfClubPro.

A Player may have:

- PlayerID

- Optional DotGolfMemberID

- Name

- Handicap index

- Equipment profile

- Historical shot record

### Club

A piece of golfing equipment used by a player.

A Club may have:

- ClubID

- Name

- Club type

- Average carry distance

- Dispersion pattern

- Confidence rating

### Course

A playable golf course.

A Course may have:

- CourseID

- Name

- Location

- Holes

- Tee sets

- Local rules

### Hole

A playable hole within a course.

A Hole may have:

- HoleID

- Hole number

- Par

- Stroke index

- Tee locations

- Green location

- Hazards

- Ideal landing areas

### Round

A playing session by a player on a course.

A Round may have:

- RoundID

- PlayerID

- CourseID

- Start time

- Finish time

- Selected tee set

- Holes played

- Shots recorded

### Shot

A recorded stroke or ball movement event.

A Shot may have:

- ShotID

- RoundID

- HoleID

- ClubID

- Start location

- End location

- Distance

- Direction

- Lie

- Weather context

- Penalty context

### Recommendation

Advice generated by the AI Caddy.

A Recommendation may include:

- Suggested club

- Aim point

- Wind adjustment

- Confidence score

- Coaching explanation

## Identifier Strategy

GolfClubPro uses strongly typed identifiers:

- PlayerID

- DotGolfMemberID

- CourseID

- HoleID

- ClubID

- RoundID

- ShotID

External identifiers are kept separate from internal identifiers.

## Domain Boundaries

GolfCore should not directly depend on:

- SwiftUI

- WeatherKit

- CloudKit

- SwiftData

- Core Location

Adapters and services may translate Apple framework types into GolfCore domain types.

## Future Domain Areas

- Handicap calculation

- Course mapping

- Shot dispersion

- Strokes gained

- Weather-adjusted carry

- AI learning model

- Coach sharing
