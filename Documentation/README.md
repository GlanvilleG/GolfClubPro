
# GolfClubPro Documentation

## Purpose

This documentation set is the authoritative reference for GolfClubPro and the GolfCore package.

It defines the product intent, architecture, domain model, engineering standards, design decisions, and delivery approach.

## Documentation Principles

- Documentation evolves with the code.

- Significant decisions are recorded as ADRs.

- Domain language is defined before implementation.

- Architecture documents explain why decisions were made.

- Code, tests, and documentation must remain aligned.
